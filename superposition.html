Principio di Sovrapposizione

Definizione

In un circuito lineare con più sorgenti indipendenti, la risposta (tensione o corrente) in qualsiasi parte del circuito è uguale alla somma algebrica delle risposte causate da ciascuna sorgente agente indipendentemente.

Vtotale = V1 + V2 + V3

Procedura per 3 sorgenti:

  1. Spegnere tutte le sorgenti tranne una (sostituire le sorgenti di tensione con cortocircuiti e le sorgenti di corrente con circuiti aperti)
  2. Calcolare la risposta del circuito per la sorgente attiva
  3. Ripetere i passi 1-2 per ciascuna sorgente
  4. Sommare algebricamente tutte le risposte parziali

Esempio con 3 Sorgenti

Circuito Originale

Circuito con 3 sorgenti indipendenti

Dati:

  • V1 = 12V
  • V2 = 9V
  • I1 = 2A
  • R1 = 4Ω
  • R2 = 6Ω

Analisi Passo-Passo

1. Solo V1 attiva

(V2 cortocircuitata, I1 aperta)

V'R2 = 12V × (6Ω)/(4Ω+6Ω) = 7.2V

2. Solo V2 attiva

(V1 cortocircuitata, I1 aperta)

V''R2 = 9V × (4Ω)/(4Ω+6Ω) = 3.6V

3. Solo I1 attiva

(V1 e V2 cortocircuitate)

V'''R2 = 2A × (4Ω∥6Ω) = 2A × 2.4Ω = 4.8V

Risultato Finale:

VR2 = 7.2V + 3.6V + 4.8V = 15.6V

Problemi Pratici

Problema 1: Due Sorgenti di Tensione

Circuito con:

  • V1 = 10V, V2 = 5V
  • R1 = 2Ω, R2 = 3Ω
V' = 10V × 3Ω/5Ω = 6V
V'' = 5V × 2Ω/5Ω = 2V
VR2 = 6V + 2V = 8V

Problema 2: Tensione e Corrente

Circuito con:

  • V1 = 12V, I1 = 3A
  • R1 = 4Ω, R2 = 4Ω
V' = 12V × 4Ω/8Ω = 6V
V'' = 3A × (4Ω∥4Ω) = 6V
VR2 = 6V + 6V = 12V

Applicazioni nelle Telecomunicazioni

Analisi di Circuiti RF

Utilizzato per analizzare circuiti a radiofrequenza con molteplici sorgenti di segnale.

Progettazione di Amplificatori

Aiuta a separare i contributi di diverse sorgenti in stadi amplificatori complessi.

Sistemi di Potenza

Applicato nell'analisi di sistemi di alimentazione con molteplici generatori.

Analisi di Interferenze

Permette di valutare separatamente i contributi di diverse sorgenti di interferenza.
